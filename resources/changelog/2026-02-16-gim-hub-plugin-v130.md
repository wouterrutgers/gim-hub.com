# GIM Hub plugin v1.3.0

- Added support for the new bank **Deposit containers** flow.
- Added support for **Coal bag**, **Rune essence pouches**, **Fish barrel/Fish sack barrel**, **Plank sack**, **Master scroll book**, and **Tackle box**.
- Reworked item transfer inference across inventory/equipment/bank for more accurate deposit and withdraw tracking.
- Added better handling for **Deposit X/Withdraw X** and default quantity modes in both bank and deposit box interactions.
